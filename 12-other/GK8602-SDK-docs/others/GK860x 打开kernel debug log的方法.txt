GK860x 打开kernel debug log的方法：
(Edited By 邓海绍 on 2016-08-22，if have any problems, pls contact me.)

1. PC通过串口连接开发板，按住PC的ENTER键的同时，给开发板上电，可以进入 uboot 的shell模式；

2. 输入 printenv 可以看到当前使用的 环境变量，

3. 输入 editenv bootargs

4. 修改 switch_printk=0 为 switch_printk=1，回车；

5. 输入 saveenv 保存修改，重启就可以看到 kernel debug log了；
